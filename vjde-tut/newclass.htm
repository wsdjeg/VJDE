新建一个类

我们现在需要一个 HelloWord 的程序用来完成我们的开发任务．通常，我们可以用两种方式来创建这个类．
  1. 使用向导 :VjdeNclass来创建一个类， 或者:VjdeNmain 来创建一个包括一个 main　函数的类
  2. 使用 :e Test.java 来创建这个类.^_^
使用向导创建的类，可能是如下的样子:

/*
 * Test.java 1.00 Wed Aug 30 16:05:57 中国标准时间 2006
 *
 * 版权信息
 */

package test;

public class Test {
        public static void main(String[] args) {
        }
}
// vim: ft=java
这就是由代码向导功能创建的类．这个向导的模板是可以自定义和修改的，这将在后面详细解释．
如果我们使用了工程管理，并指定了g:vjde_src_path,则 test/Test.java 会生成在指定的目录下,如:src/test/Test.java. 否则,将会使用当前目录生成目录结构.
好了,将以 Test.java 为例,把 VJDE 的魅力一点点向你展开.

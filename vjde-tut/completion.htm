代码提示

现在,我们有一个Test.java , 我们希望在里面输出一个 Hello World 作为我们的第一个开发计划.^_^,好像目标有点小.
基本上,我们知道,有个System
因此,我们需要输入 System. 然后呢,我们希望能看到提示,这时输入ctrl+x ctrl+u,有如下屏幕:
在这里,我们可以用:
ctrl+n 选择下一条
用 ctrl+p 选择上一条
用 ctrl+y 确认当前选择的作为输入
ctrl+e 拒绝本次菜单
小提示 在没有菜单表示选中时,这里可以用键盘输入进行过滤.要让菜单不选择,菜单显示出来后直接使用ctrl+p 即可.
同样,当我们用二次 ctrl+n,一次 ctrl+y 选择了out 这个变量的时候,再输入一个 . ,现在,输入行变成了:
System.out.
再次使用 ctrl+x ctrl+u 的代码提示功能:

到此,我们已经知道如何提示函数,成员变量了.
小提示如果你只是想提示选择一个本类中的成员变量或者变量,或者函数,你不需要这么复杂的功能,使用 ctrl+x ctrl+p / ctrl+x ctrl+n 就可以了.
下面,我们再介绍如何提示 package. 输入 import java.后,就可以使用 ctrl+x ctrl+u 进行提示了.

输入 import java.util.后,就可以使用 ctrl+x ctrl+u 进行提示了.

或者,在编辑过程中,直接输入 java.util. 用 ctrl+x ctrl+u 提示,选择你喜欢的类,如 Vector

此时,编辑界面就是 
			java.util.Vector
在这种情况下,我推荐大家再使用一个命令,移动光标到Vector上,使用命令\je 这是一个通常模式下的命令,把代码格式化如下: 
import java.util.Vector;
...
	Vector
好了,代码提示就说到这里.
提示:如果你直接在一个括号后面直接使用 ctrl+x ctrl+u ,你将得到这个函数的参数提示.

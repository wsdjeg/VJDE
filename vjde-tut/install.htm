安装

1. 软件需求:
  • Vim 7.0 以上版本
  • VJDE 最新版本
  • ruby 1.8.0 以后版本(这主要是用于代码模板功能和向导功能,如果不使用这两功能，可以不安装)
2-a 软件下载: (windows 用户)
  1. 从VIM 主站下载7.0标准完整版.
  2. 从Cream 下载 最新Vim,覆盖安装7.0标准版.这样,可以使用标准版中的中文,也可使用最新版的补丁.以后升级只要升级本部分即可.
  3. 从下载 ruby for Windows.
  4. 从下载 VJDE　最新版
2-b 软件下载，(Linux 用户)
  1. 从Vim 下载站下载你适合的源代码．
  2. 如果你的系统没有 ruby ,请从Ruby 主站下载 Ruby.
  3. 从下载 VJDE　最新版
3. 我们把 $VIM 定义成为你的 VIM所在的目录，如我使用 vim70 在 d:\vim\vim70 下，则 $VIM 指 d:\vim
对于 Linux 用户 ,$VIM 可能就是 /usr/share/vim , 安装只需要把 vjde.tgz 解压缩到　$VIM\vimfiles 下即可．
tar -xzf vjde.tgz -C /usr/share/vim/vimfiles
现在，安装成功了，如果你需要使用包括了一个以上的java文件，并且引用了外部的jar 的时候，请使用工程开始你的 VJDE 之旅．
如果你只需要一两个类来做个小测试，请从新建一个类开始.

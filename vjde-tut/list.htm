命令一览

杂项命令
  • :Vjdeas {filename} 将工程另存为{filename}
  • :Vjdeload {filename} 载入工程{filename}
  • :Vjdesave 保存当前工程
  • :Vjdec 编译当前的类
  • :Vjder {class} {parameter} 运行当前类.其中{class}和 {paramter} 可以没有,表示运行当前类,无参数.
  • :Vjdegd 跳转到当前符号的定义.
编辑中常用
  • \el 将变量重构成局部变量 Extract Local variable
  • \em 将变量重构成成员变量 Extract Member variable
  • \ep 将变量重构成参数中的一个 Extract to Parameter
  • \en 将当前选择的重构成常数.在选择模式下有用,目前只对String,byte,int,double,类型有效 Extract coNst.
  • \jt 将选择的代码块用 try-catch 块包围起来. Java Try-catch block
  • \je 将当前光标下使用的全类名改成缩写,并添加包引用到文件头. Java Export import.
  • \jg 为当前光标下的变量生成get-set 函数. Java Getter-setter generater
  • \jc 为当前光标下的符号生成文档.包括函数,变量和类. Java Commenter
  • \js 将选择了的包进行排序. Java Sort import.
  • \fi 在编译错误中,修正 没有找到符号的错. Fix with Import
  • \ft 在编译错误中,将没有报告的异常用 try-catch 块捕获. Fix with Try-catch
  • \fr 在编译错误中,将没有报告的异常用 throws 语句声明. Fix with thRows
  • \ai 为当前光标下的符号添加引用.只对类有效. Add Import
  • \ri 将本类中没有使用上的 import 语句进行删除. Remove Import
  • \oe 重载函数. Override for Extends
  • \oi 实现接口. Override for Implements
  • \as 添加Singleton 模型. Add Singleton.
